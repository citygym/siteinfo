Oracle este o companie globală de tehnologie cunoscută pentru dezvoltarea și comercializarea de software de gestionare a bazelor de date, sisteme hardware și servicii cloud. Produsele sale sunt utilizate pe scară largă în întreprinderi și organizații din întreaga lume pentru gestionarea eficientă a datelor și a infrastructurii IT.

PRODUSE:

Oracle database

Oracle Database este un sistem proprietar de gestionare a bazelor de date cu mai multe modele produs și comercializat de Oracle Corporation. Este o bază de date utilizată în mod obișnuit pentru rularea încărcăturilor de lucru ale bazelor de date de procesare a tranzacțiilor online , depozitare de date și mixte . Oracle Database este disponibilă de mai mulți furnizori de servicii on-prem, on-cloud sau ca o instalare hibrid cloud. Poate fi rulat pe servere terțe, precum și pe hardware Oracle .

Oracle Enterprise Manager (OEM)

Oracle Enterprise Manager oferă instrumente de monitorizare și management bazate pe web pentru produsele Oracle (și pentru unele software de la terți), inclusiv gestionarea bazelor de date, managementul middleware, managementul aplicațiilor, managementul hardware și virtualizării și managementul cloud.

Net Beans

NetBeans este un mediu de dezvoltare integrat (IDE) pentru Java. NetBeans permite dezvoltarea aplicațiilor dintr-un set de componente software modulare numite module. NetBeans rulează pe Windows, macOS, Linux și Solaris. Pe lângă dezvoltarea Java, are extensii pentru alte limbaje precum PHP, C, C++, HTML5, și JavaScript. Aplicațiile bazate pe NetBeans, inclusiv NetBeans IDE, pot fi extinse de către dezvoltatori terți.
